### C++ practice project repo

> C++简单的小项目练习

* **`opencv-monitor`**
    
    * 基于opencv实现的实时监控和运动检测

* **`threadpool`**
    
    * 线程池实现（`C++11 `练习）

* **`circle-data-buffer`**
    
    * 环形缓冲区实现

* **`web-server-frame`**

    * 简单的web服务框架实现（`C++11 `练习）

* **`learn-linux`**

    * Linux下C++开发练习 

* **`get-info`**

    * 获取机器的big/little endian类型

* **`design-model`**

    * 基于C++的设计模式练习

* **`message-queue`**

    * C++11实现简单的消息队列

* **`smart_ptr`**

    * C++11实现智能指针

* **`producer_consumer`**

    * C++11实现生产者消费者队列
